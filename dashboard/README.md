# Dashboard

## Project Setup
```sh
npm install
```

### Development
```sh
npm dev # This will run a development server with hot-reload enabled.
```

### Build for Production
```sh
npm build
```

### Serve Production Build
```sh
npm serve [--port <port>]
```

### Type-check and Lint Code
```sh
npm lint
```

### Autofix and Format Code
```sh
npm lint
```
